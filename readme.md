# NodeMaster API

> Backend Application that is a Bootcamp directory website

## Usage

Rename "config/config.env.env to config.env and update the values/settings to your own

## Install Dependencies

```
npm instal
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
